3D Color Picker

You can pick RGB or HLS color code using RGB color cube and HSV color cone.

  • If you click any color cube on below, preview color and color codes will be shown.
  • You can zoom up/out using scrolling with mouse wheel.
  • You can change the camera angle using drag and drop (drag and drop with right-click enables you to change the position of cube or cone).
